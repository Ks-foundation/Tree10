Tree10의 주요 기능

  • 파일 시스템: 파일 및 디렉토리 생성, 삭제, 읽기, 쓰기 기능.
  • 메모리 관리: 메모리 할당(`malloc`), 해제(`free`), 재할당(`realloc`).
  • 입출력 관리: 화면에 텍스트 출력 및 사용자 입력 처리.
  • 바이너리 파일 실행: `.air` 파일을 직접 실행할 수 있는 기능.
  • 명령어 기반 쉘(`dmd`): 간단한 명령어로 운영체제와 상호작용 가능.
